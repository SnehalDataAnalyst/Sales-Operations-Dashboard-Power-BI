# Sales-Operations-Dashboard-Power-BI
A multi-page business intelligence dashboard analyzing sales performance, customer behavior, product metrics, and regional operations for BMCS — built to support executive decision-making through interactive reporting.

## 📌 Project Overview
This Power BI report provides end-to-end business analytics across four core domains: executive performance, customer intelligence, product analytics, and operational targets. It enables stakeholders to monitor KPIs, compare actuals vs. targets, and drill into granular product and customer-level insights.
